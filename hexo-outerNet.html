Hexo系列(二)之部署外网博客

发表于 2018-06-03 | 分类于 hexo | 围观次数

上一篇Hexo系列(一)之搭建本地博客，介绍了怎么搭建本地博客，但博客光在本地观看可不行呀，你肯定会想放到外网上，通过网络可以访问，可以放到服务器上！对，没错！但是把其放到服务器上需要占很大的资源，成本也非常高。那怎么办呢？没事，已经有解决方案了，可以除花时间外，几乎是零成本。不需要服务器，也不需要考虑文件存储空间的问题。就是部署在托管平台上！

安装Git

Git是目前世界上最流行的 分布式版本控制系统 。使用Git可以帮助我们把本地的网页和文章等内容提交到Gihub等托管平台上，实现同步。

下载Git版本工具,下载完之后需要安装，下面是安装步骤

找到下载好git.ext文件，鼠标双击它

鼠标双击后后，会弹出下面的窗体，点击 Next 按钮

可以选择安装路径(点击 Browse... 按钮进行选择安装路径)，也可以选择默认路径，路径确定好了之后，点击 Next 按钮

选择要安装的组件，需要安装就勾选即可（注意不要把默认要安装插件取消了哦）然后点击 Next 按钮，其中第一项是创建桌面快捷键

创建并修改开始菜单文件夹名称，勾选左下角的选项可以不创建,然后点击 Next 按钮

设置默认的编辑器，默认是选择系统的默认编辑器，点击下拉框进行选择系统中可用的编辑器选项，然后点击 Next 按钮

选择是否增加相关path环境变量，第一项是仅在内部使用，第二项是可以在电脑如dos界面使用相关命令，第三项是以上两个都，建议第二项,我选择的是第一项 Use Git from Git Bash only，然后点击 Next 按钮

选择HTTPS的传输，默认选择第一项 Use the openSSL library 即可，然后点击 Next 按钮

选择配置行结束时转换的规则，我选择的是最后一项 Checkout as-is,commit as-is ，然后点击 Next 按钮

选择使用的终端，第一项是选择默认的终端，第二个是设置为控制台也就是DOS界面，选择第一项即可，然后点击 Next 按钮

后面的都是一直 下一步(Next) 、 安装(Install) 、 完成(Finish) 等操作，使用默认项就可以了。

部署到托管平台

这里给大家介绍三个托管平台

托管平台 国内/国外 访问速度 是否支持域名 缺点
GitHub 国外 慢 支持 国内访问速度慢
oschina(码云) 国内 快 不支持 不支持自定义域名
coding 国内 快 支持 免费版访问时有广告页面

1.部署到GitHub
2.部署到coding
3.部署到oschina(码云)
